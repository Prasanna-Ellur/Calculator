# Calculator Web App

A simple and responsive calculator built using **HTML**, **CSS**, and **JavaScript**. This project performs basic arithmetic operations and provides a clean user interface with interactive buttons.

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Clear all functionality (AC)
* Backspace/Delete functionality
* Input validation to prevent multiple operators consecutively
* Responsive and modern UI
* Real-time display updates

## Technologies Used

* HTML5
* CSS3
* JavaScript 

## Project Structure

```
Calculator/
│
├── index.html
├── styles.css
└── index.js
```

## How to Run

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Open the project folder.

3. Run `index.html` in your browser.

No additional dependencies or installations are required.
